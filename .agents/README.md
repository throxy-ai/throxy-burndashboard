# Agents Starter

This starter includes portable Throxy-inspired skills and a lightweight common-mistakes baseline.

## Included folders

- .agents/skills: reusable execution workflows.
- .agents/common-mistakes: recurring pitfalls captured as examples.

## Recommended usage

1. Start with .agents/skills/coding-process/SKILL.md for implementation work.
2. Use .agents/skills/review-code/SKILL.md before opening a PR.
3. Add project-specific mistakes into .agents/common-mistakes/ as you learn.
4. Use .agents/skills/skill-creator/SKILL.md to create new skills.
